Fetch and Display Tweets and Retweets

== Description ==

A simple plugin that stores the most recent tweets as an XML file and their media in the WP media library.

Tweets are automatically updated every hour.

Nothing displayed is imported on the fly, all media and icons are loaded locally.

This is useful if you have a CSP (Content Security Policy) or use the NGX Google Page Speed module.

You can import any Twitter users public tweets.

All you require are your Twitter API keys.

In the pipeline is a plan to allow users to create customized timelines from their imported Tweets.

== Installation ==

1. Upload `api-fetch-twitter.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Twitter API details to generate your first feed.
4. Use the shortcode [api-fetch-twitter] to display your tweets.
